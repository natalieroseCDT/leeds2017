Agent-Based Model Project

About the Model

This model was created as part of an assessment for the Programming for Social Scientsts: Core Skills module at the University of Leeds (module code GEOG5995) and written in Python. The purpose of the model is to create agents that can interact with the environment they are in, as well as other agents around them. The behaviour of these agents can be controlled using an agent framework which is connected to the main model which then runs the code.

In this instance, the primary agents are sheep that can move randomly around their environment. As they move they can nibble the enviroment and add to their personal store of food. This store can then affect the speed at which the sheep move. When they have more food, they have more energy and can move faster around the environment. However, once a store becomes too large, the sheep are sick and their store goes back to zero, while the environment gains the discarded store of food.

As well as interacting with the environment, the agents can interact with each other, using a distance calculation function. When two sheep come in close proximity to one another, they can share their food store. When this happens, if an agent's store is particularly low, it is programmed to take a greater share of the combined food store.

The outputs of this model include an animated map showing the model running as well as a resultant environment file and a appended list of total stored food.

Please note that the animation only works when the output opens in a separate window
